github 格式:
  https://github.com/<user_name>/<repo_name>
    https://github.com/<user_name>/<repo_name>.git
    gh repo clone <user_name>/<repo_name>
    https://github.com/<user_name>/<repo_name>/archive/refs/heads/master.zip
      浏览器 地址栏 直接打开 即 下载
