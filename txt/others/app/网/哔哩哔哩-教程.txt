基本功:和面揉面#品诺美食开课啦
  目标:三光:手光盆光面光
  比例:
    面粉:2斤
    水:460g
  和面:
    拌面:分次加水，每次往干面粉加水，充分搅拌
    揣面:以拳压面，净手擦盆，折叠面团，重复，直至 手光盆光
  醒面:
    和面成团后，封闭醒面20分钟，为 之后再揉透揉光（把面团内结块的小疙瘩揉散）做准备，醒面结束时，面团变松好揉（我猜是因为多数面粉充分吸收水分变软）
  揉面:
    针对面团一侧，左掌压坑，右掌将坑叠回，从右到左，旋转面团换侧，重复，直至 面团极长。
      不要抓面，要用巧劲压叠
    面团太长太细，则 长度方向进行折叠，压扁，重复步骤一，直至 面团表皮细腻，用刀切开的切面平整，无生面疙瘩
      条状比圆团更有利于揉透（将中间厚芯压扁压透，两侧叠回）

蒸包子:常见问题#品诺美食开课啦
  比例:
    中筋面粉：500g =1斤
    酵母：4g
    无铝泡打粉/小苏打：4g
    白糖：2g
    猪油/植物油： 15g
    温水： 250g = 0.5斤
    ===
    酵母+白糖+温水=>唤醒酵母
      不加糖 => 酵母会消耗掉面粉中的糖分和游离淀粉，只剩支链淀粉和纤维素，包子 口感 粗糙
    无铝泡打粉+猪油+面粉
      猪油 => 包子 白、光、放凉不硬
      酵母 无需 加碱 vs 老酵头 含杂菌 发酸 需加碱
        加碱 太多 => 包子 发黄
    分次加水，和成面团
    揉光、使配料面粉均匀充分融合
    保鲜膜 密封，发面，直至 两倍体积、撕开里面是均匀网丝状
      发面耗时 与 酵母量、面团温度、湿度、氧气 相关
    发旺的面团 需再 揉光、排气
      不揉光 => 包子表皮 坑坑洼洼
      不排气 => 气孔大小不均匀
      排气 => 大气孔 分散成 小气孔，排出 二氧化碳，酵母菌 充分接触 氧气，为 二次醒发 打好基础
      酵母菌 无氧 产 酒精 => 酿酒
      酵母菌 有氧 产 二氧化碳 => 发面
    揪面剂子，用保鲜膜封闭 以防 表皮 风干
    压面剂子，边薄芯厚，上肉馅，左拇指挡馅，右手拉边叠皮成褶子，边拉边沿圆周移动，最后封包掐去面头（即：包子尖顶）
    包子 放蒸笼里，封闭暖和，二次醒发，直至 稍微变大，轻轻戳一下能恢复
    上开水锅，蒸包子用大火
      大火，蒸汽足 => 暄软蓬松
    上汽后12分钟，出笼（？不先冷却？），变大很多（两倍直径），用力捏会回弹，掰开气孔均匀，放凉不缩
  柳志天:
    说一下为啥家庭发面总是较粗的原因，酵母发酵是需要养分的，如果不加糖分给酵母提供营养，酵母会消耗掉面粉中的糖分和游离淀粉，只剩支链淀粉和纤维素，包子和馒头就会变得口感粗糙。
  小尾巴咪咪II:
    那是火太大，蒸汽太足，把发面催死了 ，上汽之后转中小火蒸 ，蒸熟关火 等五分钟回气后再揭盖 ，就不会遇冷塌陷 ，我以前蒸也发生过类似现象 和烫面包子差不多

肉馅:刀切剁砸+葱姜水及香料油分次加拌摔#品诺美食开课啦
宁缺c:
  肉馅技巧
  肉馅常见问题：有腥味、发干发硬发柴、不油润、缺乏香味。
  解决办法：葱姜水+调料油。

  制作葱姜水：
  准备葱丝+姜丝+一点点花椒（不是增加麻味，而是去腥增香），加入少许水，用力抓，把葱姜味融入水中。
  家庭传统做法是在肉馅中加入姜末+葱花。由于颗粒较大，和肉接触的不够充分，因此去腥效果差，如果为了去腥加太多会抢走肉味，拌好的馅长时间放置葱会出现异味；
  （葱姜水不仅可以解决去腥的问题，还可以使肉馅增加水份，使肉馅不发干、发硬、发柴）

  香料油制作：
  1、小茴香20g、花椒6g、八角5g、桂皮6g、草果2个（砸开）、香叶3g
  2、水中泡一泡，去除灰尘，湿润香料，避免下油锅时炸糊；（五香粉或十三香也可调馅，效果不如调料油效果好）
  3、400g熟豆油，凉油下入泡好的香料，开小火慢慢炸，不停翻动，火太大容易炸糊发苦；香料微黄、香味浓郁时出锅，倒入碗中，加盖焖凉使用

  （三个作用：1）肉馅增香；2）肉馅油润；3）锁住肉馅的水分）

  肉馅制作流程：
  1、肉去肉皮，刀剁比绞肉机打的更香，先剁后用刀背砸。
  2、调味：1斤肉配10g盐，3g胡椒粉，一点点白糖（白糖不是增加甜味，而是提鲜），30g生抽，拌匀。
  3、去腥：加入葱姜水。分次加入肉馅中，边加边顺着一个方向搅拌，把水打入肉馅中；一次一次加，一次一次搅拌，可以边加边摔打；
  4、分次加入肉馅中，边加边顺着一个方向搅拌，把水打入肉馅中；一次一次加，一次一次搅拌，可以边加边摔打；
  5、油润有香味的秘诀，调料油。调料油加入肉馅中，朝着一个方向搅拌均匀；拌好的肉馅，看上去油润有光泽，摸上去有弹性。

  肉馅的保存：保鲜膜封闭。
  肉馅的使用：
  1）包馄饨直接用；
  2）加入韭菜可包饺子；
  3）加大葱可包包子；
  4）加入酸菜可做肉饼；
  备注：肉与菜要分开处理，包的时候再混合，防止时间一长，菜出水，且菜容易变味。
