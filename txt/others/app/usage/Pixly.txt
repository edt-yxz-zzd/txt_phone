
Pixly

文件排列:
  按 文件名 排列
  忽略字母的大小写
  无视文件名中的前缀0
  无法显示中文字符（因为使用自带像素风字体？）

只有选中的区域会被修改！
  选中要修改的区域，相当于屏蔽对立区域！
  记得 点击 底部 unselect

！右上 前景色+背景色！ ->
  其左侧有一黄色小箭头，点击该箭头上方或下方的空位置，则会将箭头移至该处，调换前景色
！右上 当前工具+螺丝刀！ ->
  #设置当前工具，比如:矩形~正方？填充？
底部 工具栏 ->
  移动 白手套 ->
    移动/缩放 选中的图块，不是移动画布视图
    具体操作，与粘贴后相同，见『粘贴』
  多边形 六角形 ->
    六角形
      不知怎么用
    圆
      画 纵横轴椭圆
    椭圆
      画 斜椭圆
  多面体 立方体 ->
      不知怎么用
  选择 虚框 ->
    魔法棒
      点选外部+画 矩形 => 带缺口矩形/纵横折连线

右上 齿轮菜单 ->
  view ->
    setup mirrors
      ！自动生成镜像！
      对称轴 单指选中压住移动、双指选中压住旋转，选中可删除
        add mirrors: 对称轴先纵再横，接着斜
      缺点:
        对称轴 无法平移至 网格中间，只能偶数宽度对称
    toggle rulers
      开启 尺子
    toggle touch coordinates
      开启 纵横延长线，指压显示
    toggle artboard controls
      打开 移动、缩放
      此移动为画布视图的移动，与工具栏中的移动不同
      等价于 双指平移、双指缩放
    toggle mini view
      关掉此窗口
  select ->
    invert
  color ->
    replace
  edit ->
    resize canvas
    cut/copy/paste
      粘贴时注意取消『选择』
      粘贴后:
        主框内 指压移动
        四角小方格 缩放
        四角小弧 旋转
    rotate/mirror
      选择+旋转 # 镜像 可另用 view->setup mirrors
