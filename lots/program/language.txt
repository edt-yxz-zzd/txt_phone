In fact, I think everyone should learn an Imperative OO language (Java, C++, Python whatever), a functional language (haskell, erlang, OCaml whatever), a concatenative language (Factor, Forth, Cat, Joy etc), a logic programming language (Prolog, Mercury etc) and a dataflow language (labview, estereel, Lustre, verilog, pure data, MAX/MSP etc). This combination will show you that a) there are many radically different paradigms out there, b) some languages really are different and c) you cannot learn them all from a reference.
