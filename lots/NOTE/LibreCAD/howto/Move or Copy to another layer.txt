
Moving/copying entities from one layer to another
    http://forum.librecad.org/Moving-copying-entities-from-one-layer-to-another-td5707491.html
    1. select the entities
    2. select the target layer
    3. menu: Tools->Modify->Move/Copy
        instead of Edit->Move/Copy
    4. double click (one to choose reference point and another to choose target point)
    5. check "use current layer" in the popup-dialog
        check or not "keep original"


copy a block from one layer to another
    http://forum.librecad.org/Block-can-not-be-inserted-on-different-layers-tp5544026.html
    Edit the "block definition" and change the properties of all the entities to layer 0 & color, linetype & width "byblock"
