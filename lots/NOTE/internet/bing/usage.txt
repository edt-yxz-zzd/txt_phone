Bing 搜索语法
语法 	简介 	例子 	注意事项
contains 	只搜索包含指定文件类型的链接的网站。 	音乐 contains:wma 搜索只包含 .wma 文件链接的网站 	本人常用
filetype 	仅返回以指定文件类型创建的网页 	论文 filetype:pdf 查找以 PDF 格式创建的论文 	
inanchor:、inbody:、intitle 	指定搜索条件(定位标记、正文或标题)的网页。为每个搜索条件指定一个关键字 	inanchor:msn inbody:seo inbody:sem 查找定位标记中包含 msn、且正文中包含 seo 和 标题为sem 的网页 	
ip 	查找特定 IP 地址 的网站 	ip:217.4.249.222 搜索 IP 地址网站 	IP 地址必须是以英文句点分开的地址 , ip不分大小写
language 	返回指定语言的网页 	small日本 language:en 查看有关small日本的英文网页 	使用搜索生成器中的“语言”功能也可以指定网页的语言。
loc 或 location 	返回特定国家或地区的网页 	small日本 (loc:US OR loc:GB) 若要查看有关small日本 的美国或英国的网页 	在关键字 loc 后面直接指定国家或地区代码。若要搜索两种或两种以上语言，请使用逻辑运算符 OR 对语言分组。若要查看可用于 Bing 的语言代码列表，请参阅国家、地区和语言代码。超级常用
prefer 	着重强调某个搜索条件或运算符 ,将有关perfer后面的放到最前面 	烤鸭 prefer:北京 查找烤鸭的相关网页，但搜索内容主要限定在查找北京 	
site 	返回属于指定网站的网页。 	site:zhihu.com OR douban.com small日本 在知乎和豆瓣中搜 small日本 	若要搜索两个或更多域，请使用逻辑运算符 OR 对域进行分组。

百度 搜索语法
语法 	简介 	例子 	注意事项
intitle 	网页标题一定要涵盖后面的关键词 	small日本 intitle:美国 搜索的网站标题不一定要有日本但一定有美国 	intitle:和后面的关键词之间不要有空格。
site 	在特定站点中 搜索相关内容 	site:zhihu.com small日本 在知乎中搜 small日本 	site:后面跟的站点域名，不要带协议头http:// 且site:和站点之间，不要空格
inurl 	搜索 到的网站 的 url 一定要有关键字 	small日本 inurl:video 在查到的’small日本’中挑选 url 含有 video 的网站 	inurl:和后面的 不要空格
双引号 "" 	查询词 加上双引号"" 表示查询词 不能被拆分，在 搜索结果中必需完整出现，精确匹配。不加经过百度分析后可能会拆分 	"small日本" 查询到的网站必须含有 small日本 全部关键字 	本人最常用
减号 - 	查询词 在加-后 ,在搜索结果中去掉包含特定的关键词所有网页。 	买东西 -推广 	本人最常用
加号 + 	查询词 在加+后 ,在搜索结果中去掉不包含特定的关键词所有网页。 	买东西 +推广 	与""类似,但有不同
