
search upward
    ?<pattern><enter>
    # - search current word
    n - upward
    N - downward
search downward
    /<pattern><enter>
    * - search current word

    n - downward
    N - upward

switch direction
    "?"<enter>
    "/"<enter>

    https://superuser.com/questions/240468/how-can-i-switch-from-forward-to-backward-search-in-vim
    typing / Return or ? Return will always rerun the last search, but in the direction you want.
